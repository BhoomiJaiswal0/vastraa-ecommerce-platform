# 📖 Project Overview

**Vastraa** is a modern frontend e-commerce platform designed to deliver a seamless and visually engaging online fashion shopping experience.

The project demonstrates modern frontend engineering principles through reusable React components, responsive layouts, scalable project architecture, and optimized performance using Vite.

Built with a strong emphasis on user experience, maintainability, and clean code practices, Vastraa showcases how a production-ready frontend application can be developed using today's most widely adopted web technologies.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/BhoomiJaiswal0/vastraa-ecommerce-platform.git
```

Navigate to the project

```bash
cd "E-commerce website"
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
